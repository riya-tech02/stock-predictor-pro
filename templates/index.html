About This Project

This is a production-grade stock market prediction system using a Bi-LSTM with Attention Mechanism trained on historical S&P 500 data with 33 engineered technical indicators.

Deep Learning

Bi-LSTM + Attention

33 Features

RSI, MACD, Bollinger Bands

Production Ready

Deployed on Render

Model Architecture

• Bidirectional LSTM (128 → 64 units)
• Custom Attention Layer (128 units)
• 3-class output (Down/Neutral/Up)
• Focal loss for class imbalance

Training Data

• Dataset: S&P 500 (SPY) 2018-2025
• Training samples: 1,284
• Validation samples: 321
• 60-day sequence length

Tech Stack

• Python & TensorFlow 2.13
• FastAPI & Uvicorn
• Docker containerization
• Deployed on Render
